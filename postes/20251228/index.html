## Hello world!

Hugo + Github Pagesで個人サイトを改めて作った。ログがあれば振り返られる、をテーマとして運用していこうと考えている。

Hugo ThemesからHugo Bear Blogをほぼバニラで使うが、必要最低限の改造だけ施す。

## 最新記事一覧

トップページに最新記事一覧を表示する。index.htmlを開き、以下を追加する。

<ul class="post-list">
  {{ range first 5 (where .Site.RegularPages "Type" "posts") }}
    <li class="post-item">
      <span class="post-date">
        {{ .Date.Format "Jan. 02, 2006" }}
      </span>
      <a href="{{ .RelPermalink }}">{{ .Title }}</a>
    </li>
  {{ end }}
</ul>

## 外部リンクを新規タブで開く

Render Hook を用いる。layouts/_default/に_markup/を作成し、以下をrender-link.htmlとして置く。

1
2
3
4
5
6
7

{{ $isExternal := strings.HasPrefix .Destination "http" }}

<a href="{{ .Destination | safeURL }}"
   {{ if $isExternal }}target="_blank" rel="noopener noreferrer"{{ end }}
   {{ with .Title }}title="{{ . }}"{{ end }}>
  {{ .Text | safeHTML }}
</a>

外部リンクは新しいタブ、内部リンクは同じタブで開ける。Hugo v0.62 以降が必須。
